# Ironfish Tutorial
Minimal Spesifikasi VPS

4Core vCPU 

6/8GB RAM

160GB/200GB SSD/NVME Storage

### Install Ironfish pilih No.1

``` 
wget -q -O ironfish.sh https://api.nodes.guru/ironfish.sh && chmod +x ironfish.sh && sudo /bin/bash ironfish.sh
```
